# 🚀 Animated Portfolio Website

A modern, fully responsive **Animated Portfolio Website** built using **React.js, Tailwind CSS, and Vite**.
This project showcases my skills, projects, and experience with smooth animations and a clean UI.

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/prashantgupta2601/Animated-Portfolio.git
cd Animated-Portfolio
```

---

### 2️⃣ Install dependencies

```bash
npm install
```

---

### 3️⃣ Run the development server

```bash
npm run dev
```
